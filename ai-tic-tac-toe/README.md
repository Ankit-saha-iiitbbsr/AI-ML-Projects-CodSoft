# ai-tic-tac-toe
Tic Tac Toe AI with Minimax Algorithm using vanilla Javascript

## Demo 
[Play](https://ankit-saha-iiitbbsr.github.io/AI-TIC-TAC-TOE/)

## Install
`npm install`

## Execution
Just open index.html on a browser (ES6 browser support required)

## Tests

### Unit tests
`npm run test`

### End To End tests
`npm run cypress:open`
